Instructions

Dear participant , please read the instructions before the experiment.

There are a total of 54 tasks to be completed. After each set of 18 tasks, you are recommended to take a break of 30 seconds before continuing.

Techniques involved:
1. CTRL-C, CTRL-V

2. AutoComPaste: Type the phrase given in the text editor and select the auto completed suggestions given as you type.

For sentence or paragraph, after autocompaste the first sentence, you can press the right arrow keyboard button to autocomplete the next sentence and the left arrow keyboard button to remove the sentence.

For each task, you are required to copy and paste a particular phrase/sentence/paragraph in the red box using the technique stated in the instructions into the text editor and clicked 'Submit' to go to the next task.
